# About Me

## 簡介
現為台大電機系大四學生，輔修歷史系與法律系，因為跟電子學八字不合（卻因此培養出良好的自學能力），所以決定專心朝向資訊方面發展，目前感興趣的領域是嵌入式系統、資安與區塊鏈，將來可能長出機器學習的技能樹。除了電機系的相關知識，也有歷史分析與法律知識，能從不同的觀點看待事情。雖然在校成績不是名列前矛，但對於感興趣的知識掌握良好，對於程式與網路較為擅長，也能根據自己的需求做出簡單裝置。
